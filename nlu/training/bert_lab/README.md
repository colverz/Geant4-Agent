# BERT Lab

This folder is an isolated starting point for training an encoder-only model to classify
structure types from natural language. It uses the synthetic JSONL data generated by
`bert_lab_data.py`.

## Requirements
- Python >= 3.10
- PyTorch (GPU recommended if you have NVIDIA drivers)
- Transformers (HuggingFace)

## Quick Start

Generate synthetic data:

```powershell
python bert_lab_data.py --out data/bert_lab_samples.jsonl --n 2000 --seed 7 --noise_level none --unknown_rate 0.15
```

Train a structure classifier:

```powershell
python train_structure.py --data data/bert_lab_samples.jsonl --outdir models/structure --model distilbert-base-uncased --epochs 3
```

This produces a small classifier in `models/structure/`.

## Parameter Extraction (NER)

Generate data with spans:

```powershell
python bert_lab_data.py --out data/bert_lab_ner_samples.jsonl --n 2000 --seed 7 --with_spans --noise_level none
```

Train a token classifier:

```powershell
python train_ner.py --data data/bert_lab_ner_samples.jsonl --outdir models/ner --model distilbert-base-uncased --epochs 3
```

## Inference

Infer structure + params from text:

```powershell
python infer.py --text "ring of 12 modules, radius 40 mm, module 8x10x2, clearance 1 mm" --device cuda
```

## End-to-End Demo

```powershell
python end_to_end.py --text "ring of 12 modules, radius 40 mm, module 8x10x2, clearance 1 mm" --device cuda --top_k 3 --autofix --prompt_format json_schema
```

## LLM Data Synthesis (Ollama)

Generate JSONL samples using Ollama:

```powershell
python ollama_data.py --out data/bert_lab_ollama_samples.jsonl --n 50 --config configs/ollama_config.json
```

## Ollama API (Config Placeholder)

If you want to route natural language through Ollama, edit:
- `nlu/bert_lab/configs/ollama_config.json`

Minimal usage example:

```powershell
python - << 'PY'
from nlu.bert_lab.ollama_client import chat
resp = chat("Describe a ring of 12 modules with radius 40 mm.")
print(resp.get("response", "")[:200])
PY
```

## Optional: Install GPU PyTorch

If you have an NVIDIA GPU, install CUDA-enabled PyTorch (replace `cu121` if needed):

```powershell
.\.venv\Scripts\python -m pip uninstall -y torch
.\.venv\Scripts\python -m pip install torch --index-url https://download.pytorch.org/whl/cu121
```

Verify GPU is visible:

```powershell
.\.venv\Scripts\python - << 'PY'
import torch
print(torch.__version__)
print('cuda available:', torch.cuda.is_available())
print('device:', torch.cuda.get_device_name(0) if torch.cuda.is_available() else None)
PY
```
