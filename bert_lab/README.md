# BERT Lab

This folder is an isolated starting point for training an encoder-only model to classify
structure types from natural language. It uses the synthetic JSONL data generated by
`bert_lab_data.py`.

## Requirements
- Python >= 3.10
- PyTorch (GPU recommended if you have NVIDIA drivers)
- Transformers (HuggingFace)

## Quick Start

Generate synthetic data:

```powershell
python bert_lab_data.py --out bert_lab_samples.jsonl --n 2000 --seed 7 --noise_level none
```

Train a structure classifier:

```powershell
python train_structure.py --data bert_lab_samples.jsonl --outdir bert_model --model distilbert-base-uncased --epochs 3
```

This produces a small classifier in `bert_model/`.

## Parameter Extraction (NER)

Generate data with spans:

```powershell
python bert_lab_data.py --out bert_lab_ner_samples.jsonl --n 2000 --seed 7 --with_spans --noise_level none
```

Train a token classifier:

```powershell
python train_ner.py --data bert_lab_ner_samples.jsonl --outdir bert_ner_model --model distilbert-base-uncased --epochs 3
```

## Inference

Infer structure + params from text:

```powershell
python infer.py --text "ring of 12 modules, radius 40 mm, module 8x10x2, clearance 1 mm" --device cuda
```

## End-to-End Demo
